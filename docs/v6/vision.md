# V6 — Athena Home

## Objetivo

Convertir Garmin Stats en una experiencia centrada en Athena.

La Home V6 no será un dashboard tradicional. Será una pantalla de decisión diaria donde el usuario entienda en menos de 5 segundos:

- Cómo está.
- Qué debería hacer hoy.
- Por qué Athena recomienda eso.
- Cómo evoluciona su entrenamiento.

## Principio principal

Athena habla primero.  
Los datos justifican después.

## Filosofía

Antes:

Datos → Interpretación

Ahora:

Athena → Recomendación → Evidencias → Datos
